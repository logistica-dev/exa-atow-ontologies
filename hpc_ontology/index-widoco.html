Exa-AToW HPC Ontology Specification

Abstract

The Exa-AToW HPC Resource Ontology is a semantic framework designed to model and manage the complex landscape of High-Performance Computing (HPC) infrastructure. As national computing centers increasingly diversify their architectures—integrating heterogeneous compute nodes, multi-layered storage systems, and specialized accelerators—standardizing the representation of these resources becomes critical. This ontology provides a structured, machine-readable model of HPC hardware, software environments, and administrative configurations. By leveraging Semantic Web technologies, Exa-AToW enables interoperability, automated resource discovery, and intelligent querying across disparate HPC centers, ultimately facilitating more efficient resource utilization and analysis.

For an interactive exploration of the ontology structure and its associated instances, please visit our integrated Exa-AToW Ontology Viewer (onto-viewer).

Introduction back to ToC

Modern HPC systems have evolved into intricate ecosystems characterized by a wide array of processor architectures, specialized hardware accelerators, and high-performance parallel file systems. Managing information about these resources is often hampered by silos, as documentation is frequently scattered across disparate web pages, wikis, and configuration files, often in inconsistent formats. The Exa-AToW Ontology project aims to bridge this gap. By mapping the relationships between physical components, logical partitions, and software stacks, Exa-AToW creates a unified knowledge graph of national-scale HPC resources. This project provides a foundation for developers and researchers to query system capabilities, track infrastructure evolution, and automate the synchronization of hardware specifications across national computing centers.

Exa-AToW HPC Resource Ontology: Description back to ToC

Ontology for modeling high-performance computing resources, infrastructure, and physical characteristics within the Exa-AToW project. This ontology describes the physical and logical components that constitute HPC systems, including compute nodes, processors, storage, interconnects, and their relationships. It supports the modeling of supercomputer architectures, partitions, and execution environments, with a focus on French national HPC centers (IDRIS/Jean Zay, CEA/TGCC, Cines/Adastra).
